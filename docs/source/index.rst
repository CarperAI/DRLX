Welcome to DRLX's documentation!
================================
DRLX (Diffuser Reinforcement Library X) is a library made to simplify the training of diffusion models with reinforcement learning, powered by `diffusers <https://github.com/huggingface/diffusers>`_ and `accelerate <https://github.com/huggingface/accelerate>`_.
